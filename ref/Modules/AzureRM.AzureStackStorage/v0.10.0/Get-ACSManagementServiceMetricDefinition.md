# Get-ACSManagementServiceMetricDefinition

## SYNOPSIS

## SYNTAX

```
 [[-SubscriptionId] <String>] [[-Token] <String>] [[-AdminUri] <Uri>] [-ResourceGroupName] <String>
 [-SkipCertificateValidation] [<CommonParameters>]
```

## DESCRIPTION
