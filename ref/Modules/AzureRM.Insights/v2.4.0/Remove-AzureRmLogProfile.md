# Remove-AzureRmLogProfile

## SYNOPSIS

## SYNTAX

```
Remove-AzureRmLogProfile -Name <String> [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

## DESCRIPTION
The **Remove-AzureRmLogProfile** cmdlet removes a log profile.
