# Set-AzureSqlServer

## SYNOPSIS
Modifies properties of an Azure SQL Database server.

## SYNTAX

```
Set-AzureSqlServer [-ServerName] <String> [-SqlAdministratorPassword <SecureString>]
 [-Tags <System.Collections.Generic.Dictionary`2[System.String,System.String]>] [-ServerVersion <String>]
 [-Force] [-ResourceGroupName] <String> [-Profile <AzureProfile>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The Set-AzureSqlServer cmdlet modifies properties of an Azure SQL Database server.
